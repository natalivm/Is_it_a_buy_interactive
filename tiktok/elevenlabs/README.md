# ElevenLabs — готові тексти для озвучки

Чисті наративи для TTS: лише текст + теги пауз `<break time="Xs" />` (ElevenLabs їх **не читає**
вголос, а робить паузу). Без емодзі й розмітки. Цифри записані так, щоб коректно читались українською
($ → «доларів», % → «відсотків», ×2.5 → «у 2,5 раза»).

Файли: `asts.txt`, `nbis.txt`, `amat.txt`, `sezl.txt`, `be.txt`, `stx.txt`, `smh.txt`.

## Як користуватись
1. **Модель:** Multilingual v2 (підтримує українську + теги `<break>`). Встав текст як є.
2. **Паузи:** `<break time="0.5s" />` — від 0.3 до 0.7 с між думками; до 3 с максимум. Не став
   надто багато поспіль — може зʼявитись артефакт. Альтернатива — трикрапка «…» дає легку паузу.
3. **Якщо модель v3:** заміни `<break .../>` на трикрапки «…» або тег `[pause]` (v3 краще реагує
   на пунктуацію, ніж на SSML-паузи).
4. **Налаштування голосу (орієнтир):** Stability ~50, Similarity ~75, Style — низький; за потреби
   підніми Style для більшої «подачі».
5. **Перевір числа:** якщо голос десь читає число неправильно — впиши його словами
   (напр. «сто сім» замість «107»).
