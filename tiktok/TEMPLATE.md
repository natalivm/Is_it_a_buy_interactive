# $TICKER — TikTok-сценарій (шаблон, компакт)

Один звʼязаний абзац на акцію (без розбивки 1.2.3). Приклади: `asts.md`, `nbis.md`, `amat.md`, `sezl.md`.

## Принципи
1. **Хук на старті:** «Чому [TICKER] і саме сьогодні?» — незвичний рух, свіжий каталізатор, різкий %.
2. **Один наратив:** речення чіпляються одне за одне (Коротко… → Але… → Тому… → Головне…).
3. **Технічне обґрунтування:** не просто рівень, а ЧОМУ. Патерн/індикатор → висновок
   (напр. «подвійне дно на $66 встояло → розворот», «$107 = геп + Bollinger → стіна», «RSI 86 → перекуп»).
4. **Дуга:** хук → що це → рух (ралі/обвал, %) → чому зараз/каталізатор+дата → рівні з технічкою →
   ціль (умова пробою) → горизонти (свінг vs модель) → сигнали (тренд vs імпульс) → план по кроках → правило.
5. **Стиль:** «ти», без води, точні цифри зі слайдів; кінець — «Не фінансова порада».
