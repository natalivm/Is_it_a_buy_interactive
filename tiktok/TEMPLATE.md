# $TICKER — TikTok-сценарій (шаблон)

Шаблон коротких меседжів під слайди презентації. Мета: **єдиний наратив**, який читається
за раз, з хуком на старті й чіткою думкою в кінці. Дивись готовий приклад: `tiktok/asts.md`.

## Принципи

1. **Вступ = хук (2–3 сек).** Відповідай на «чому саме ця акція і саме сьогодні?». Створи
   інтригу (незвична ситуація, свіжий каталізатор, різкий рух), а не суху назву компанії.
2. **Звʼязаний наратив.** Кожен меседж підхоплює попередній — використовуй містки:
   *Спершу… → А потім… → Але… → Що штовхне далі? → Тепер попереду… → Пробʼє/ні… → І тут увага…
   → Тому… → Головне…*. Прочитані підряд, рядки = одна історія.
3. **Стиль.** Звертання на «ти», короткі речення, 1 рядок = 1 думка, 0–1 емодзі на рядок.
4. **Дуга розбору:** гачок → що це → рух (ралі/обвал) → чому зараз → каталізатор → рівні →
   ціль → горизонти (свінг vs інвестиція) → сигнали → план по кроках → правило/CTA.
5. **Кінцівка.** Заклик до дисципліни + «зберігай собі» + дисклеймер *(не фінансова порада)*.
6. **Точність.** Цифри мають збігатися зі слайдами (ціна, рівні, %, дати).
