## Голос і структура
1. **Відкриття (завжди):** «Сьогодні розглянемо технічний аналіз по акції [TICKER]…»
2. **Інклюзивний тон:** «у нас», «нам», звертання на «ти». Розмовно, без канцеляриту.
3. **Дуга:** що сталося (падіння/ралі + патерн) → чому відбій/рух (новина/каталізатор + дата) →
   рівні з технічним обґрунтуванням (патерн/індикатор → висновок) → ціль з умовою пробою (+%) →
   горизонт (свінг vs 5-річна модель) → тренд проти імпульсу → правило.
4. **Технічка вплетена:** «подвійне дно на $66 встояло», «$107 = геп + Bollinger, тому опір»,
   «RSI 86 → перекуп», «тижневі 9>50>200 → тренд бичачий».
5. **Фінал (завжди):** «Головне — [правило/терпіння б'є хайп]. А щоб отримувати огляди вчасно —
   заходь до нас у телеграм-групу 👉 t.me/market_predictions». Далі — *(не фінансова порада)*.
